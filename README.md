## ✨ Sobre o projeto

O **Images to Sprit Sheet PRO** é uma ferramenta feita em **HTML, CSS e JavaScript** que transforma uma sequência de imagens em uma única **spritesheet**.

Ela também permite exportar um arquivo **JSON** com as posições dos frames, facilitando o uso em:

🎮 jogos
🤖 animações com IA
🧙 personagens
💥 efeitos visuais
🧪 protótipos
🎞️ sequências de frames
🖼️ organização visual de imagens

## 🌍 Publicar com GitHub Pages

Para deixar a ferramenta online:

1. 📤 Envie o arquivo `index.html` para a raiz do repositório.
2. ⚙️ No GitHub, vá em **Settings**.
3. 📄 Entre em **Pages**.
4. Em **Build and deployment**, selecione:

   * **Source:** Deploy from a branch
   * **Branch:** `main`
   * **Folder:** `/root`
5. 💾 Clique em **Save**.
6. 🚀 Depois de publicado, a página ficará em um endereço parecido com:


## 🎯 Para que serve

Esta ferramenta serve para juntar várias imagens em uma única **spritesheet**.

Isso é útil para criar e organizar frames de animação em uma grade única, principalmente em projetos de:

| Uso             | Exemplo                                    |
| --------------- | ------------------------------------------ |
| 🎮 Jogos        | personagens, ataques, movimentos           |
| 🤖 IA           | frames gerados por inteligência artificial |
| 🎞️ Animações   | sequências quadro a quadro                 |
| 💥 Efeitos      | explosões, magias, partículas              |
| 🧪 Protótipos   | testes visuais rápidos                     |
| 🖼️ Organização | imagens agrupadas em um único arquivo      |

## ⚙️ Recursos disponíveis

A ferramenta permite configurar:

* 🔢 quantidade de linhas;
* 🔢 quantidade de colunas;
* 🖼️ número máximo de imagens;
* ↔️ espaçamento entre frames;
* 📐 largura final;
* 📏 altura final;
* 🎞️ intervalo de seleção dos frames;
* 📦 exportação em PNG;
* 🧾 exportação em JSON.

---

## 🚀 Como usar

1. 📁 Clique em **Escolher arquivos**.
2. 🖼️ Selecione as imagens que deseja transformar em spritesheet.
3. ⚙️ Ajuste as linhas, colunas, tamanho final e espaçamento.
4. ▶️ Clique em **Gerar Sprit Sheet**.
5. 👀 Confira o resultado no canvas.
6. 📥 Clique em **Download PNG** para baixar a spritesheet.
7. 🧾 Clique em **Download JSON** para baixar o mapeamento dos frames.

## 🧠 Observação sobre o README.md

Não cole todo o código HTML dentro do `README.md` esperando que o GitHub mostre a ferramenta funcionando.

O GitHub remove ou bloqueia partes de:

* `<script>`
* CSS avançado
* JavaScript
* eventos interativos
* código que possa afetar a segurança da página

✅ A forma correta é:

```text
index.html   → código completo da ferramenta
README.md    → apresentação, instruções e documentação
```
